# node-notes
Node Command Line Application for **Notes**

## Getting started:

App requires [Node.js](https://nodejs.org/) v12+ to run.

```
$ npm install
```

Available commands:

```
$ node app add --title="List" --body="Items"
$ node app list
$ node app read --title="List"
$ node app remove --title="List"
```
